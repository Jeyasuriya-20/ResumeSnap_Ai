🚀 ResumeSnap AI
An AI tool that scans your resume and gives you a score, feedback, and free course suggestions.

What It Does
- Scores your resume out of 10
- Checks ATS readiness
- Shows your strengths and weak areas
- Suggests free courses to improve
- Gives an AI roast 

Built With
- Python
- Streamlit
- Groq AI (LLaMA 3.3)
- pdfplumber
How To Run :
Bash
 pip install streamlit pdfplumber python-docx grop streamlit run app.py

How To Use :
- Select Fresher or Experienced
- Paste the Job Description
- Upload your Resume (PDF or DOCX)
- Click Analyze button 
- See your results 

API Key :
Get a free key from console.groq.com
Add it as environment variable: GROQ_API_KEY
